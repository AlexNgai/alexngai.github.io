Machine Learning

01. Mouse BMI Project

MEng research project to create a viable long-term brain machine interface (BMI) using noninvasive optical sensing. Transgenic mice neurons are modified to produce visible fluorescent signals. Awake running mice are then recorded to predict neural intentions of limb position and movement using statistical classifiers.

link

02. sick.ly

A submission to the 2014 Koding Global Hackathon. Uses data from the Twitter streaming API to track the propagation of sickness throughout the world by classifying different types of sickness in tweets.
